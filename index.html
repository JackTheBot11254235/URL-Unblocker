URL Unblocker URL Unblocker is a project which allows you to view sites that may be blocked by your ISP or employer. I am currently still working on it and is quite buggy. It has a Captcha(to prevent spam) and it also logs every request you make through it. It unblocks the URL you enter by the server viewing the webpage, not you, and then the server transfers the data back to you. Some web-pages do not display correctly because of images or css being linked without the full website address at the beginning of the links. Work in Progress, still quite buggy.
