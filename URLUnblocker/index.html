This tool allows you to view web pages that may be blocked by your ISP or employer. It works by getting the server to view the web page you specify and then transfers the data to you. Please do not use this for illegal purposes or spam it. Thank you.

All use of this site is logged. The URL you request, your IP address, and the date & time of your request is logged. We will not share this data with any other party unless required to do so by law.

Please note that this only works with single pages. Clicking on a URL or button on a page viewed by using this tool will NOT work. Please use the direct URL instead.
